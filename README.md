# Tiny Twitter Api

## To install
#### git clone https://github.com/salmazz/tiny_twitter.git
##### composer install
##### cp .env.example .env
##### php artisan key:generate
## To make migration and seed data run this command
##### php artisan migrate --seed
## To make test run this command and show test cases
#### php artisan test
